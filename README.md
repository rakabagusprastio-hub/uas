DESKRIPSI APLIKASI
Aplikasi ini merupakan program berbasis Python (console) yang digunakan untuk membuat invoice penjualan secara sederhana.  
Program dapat menerima input data barang, menghitung total harga, PPN 10%, dan menampilkan invoice secara otomatis.

 ALUR APLIKASI
1. Program menampilkan header invoice
2. User memasukkan jumlah jenis barang
3. User menginput:
   - Nama barang
   - Satuan
   - Kuantum
   - Harga satuan
4. Program menghitung jumlah per barang
5. Program menghitung subtotal
6. Program menghitung PPN 10%
7. Program menampilkan grand total dan informasi pembayaran
